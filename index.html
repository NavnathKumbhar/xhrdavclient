WebDAV Client ajax API using google closure library(UNFIXED)

WebDAV Client Ajax API library using google closure library

Dependencies

Closure Library

Install

1. Integrate and compile scripts 

    # Integrate scripts
    $ cd unzip OR untar directory
    $ ./tools/builder.sh
    => xhrdavclient.js       # Generate integrated scripts

    # Integrate, compile and optimize scripts
    $ ./tools/builder.sh -s     # SIMPLE_OPTIMIZATIONS
    $ ./tools/builder.sh -w     # WHITESPACE TRIMING ONLY
    => xhrdavclient.js       # Generate compiled scripts

    # Release build scripts(goog.DEBUG=off)
    $ ./tools/builder.sh -sr
    => xhrdavclient.js       # Generate compiled scripts

2. Put the generated script in Application.

3. Configuration 

    <script src="xhrdavclient.js" type="text/javascript"></script>
    <script type="text/javascript">
      goog.require('xhrdav.lib.DavFs');
    </script>
